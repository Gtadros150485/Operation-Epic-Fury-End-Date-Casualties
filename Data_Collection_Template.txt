1. Collect data points 1-3 (critical) every day
2. Update model with actual vs predicted weapons usage
3. Recalculate remaining inventory
4. Adjust production capacity based on facility strikes
5. Update strategic objectives completion percentage
6. Regenerate duration prediction

Model Refinement:
- If actual weapon usage > predicted: Conflict will end SOONER
- If facility destruction > expected: Production capacity drops, conflict ends SOONER
- If objectives completed faster: War ends on objectives timeline
- If resupply detected: Extend duration estimate

Validation Thresholds:
- Day 4-7: Model accuracy within ±20% → Continue using model
- Day 4-7: Model error > 30% → Recalibrate with new data
- Day 8+: Weekly recalibration with cumulative data
